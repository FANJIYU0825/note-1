##mod運算的性質

交換律  
(w + x) mod n = (x + w) mod n
(w * x) mod n = (x * w) mod n

結合律  
[(w + x) + y] mod n = [w + (x + y)] mod n
[(w * x) * y] mod n = [e * (x * y)] mod n

分配律  
[w * (x + y)] mod n = [(w * x) + (w * y)] mod n
[w + (x * y)] mod n = [(w + x) * (w + y)] mod n

單位元  
(0 + w) mod n = w mod n
(1 * w) mod n = w mod n
